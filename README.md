# bird_exporter Helm Repository

This branch hosts the Helm chart repository for bird_exporter.

## Usage

```bash
helm repo add bird_exporter https://czerwonk.github.io/bird_exporter
helm repo update
helm install my-bird-exporter bird_exporter/bird_exporter
```

For chart source and documentation, see the [main branch](https://github.com/czerwonk/bird_exporter/tree/main/charts/bird-exporter).
